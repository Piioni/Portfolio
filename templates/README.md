# Content Templates

Usá estas plantillas como base para crear/editar contenido.

- `project.template.md`
- `post.template.md`
- `experience-item.template.yml`
- `skill-item.template.yml`
- `interest-item.template.yml`

Después de editar contenido, corré:

```bash
bun run lint
bun run typecheck
bun run test
```
